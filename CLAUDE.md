## Build & Tool Commands
- **初始化專案**: `uv init`
- **安裝開發工具**: `uv tool install "cookiecutter==1.7.0"`
- **執行專案生成**: `cookiecutter <template>`
- **同步依賴環境**: `uv sync`
- **執行 Python 指令/腳本**: `uv run <command_or_script>`

## Code Style & Development Guidelines
- 依賴管理與執行一律採用 `uv`。
- 終端機指令執行前需載入 `mise` 環境變數。
